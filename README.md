# GUI based Dictionary

**GUI based Dictionary** is a Python application developed using Tkinter for the GUI. This application allows users to search for word meanings, add new words, update existing words, delete words, and print the list of words and their meanings. It uses the `PyDictionary` library for retrieving word meanings.

## Features

- **Search Meaning**: Look up the meaning of words.
- **Add Word**: Add new words with their meanings to the dictionary.
- **Update Word**: Update the meanings of existing words.
- **Delete Word**: Remove words from the dictionary.
- **Print List**: Print the current list of words and their meanings.
- **GUI Interface**: User-friendly graphical interface built with Tkinter.

## Technologies Used

- **GUI**: Tkinter
- **Backend**: Python
- **Dictionary API**: PyDictionary
- **Image Processing**: PIL (Pillow)

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/shraddacvn/gui-based-dictionary.git
    ```
2. Navigate to the project directory:
    ```bash
    cd gui-based-dictionary
    ```
3. Install the required libraries:
    ```bash
    pip install -r requirements.txt
    ```
4. Run the application:
    ```bash
    python gui_based_dictionary.py
    ```

## Usage

1. Run the application by executing `gui_based_dictionary.py`.
2. Enter the word you want to search for in the input field and click "SEARCH MEANING" to get the meaning.
3. To add a new word, enter the word in the input field, type its meaning in the text area, and click "ADD WORD".
4. To update an existing word's meaning, enter the word in the input field, type the new meaning in the text area, and click "UPDATE WORD".
5. To delete a word, enter the word in the input field and click "DELETE WORD".
6. Click "PRINT LIST" to display the current list of words and their meanings.
7. Click "EXIT DICTIONARY" to close the application.

## File Descriptions

- **gui_based_dictionary.py**: The main script to run the application.
- **requirements.txt**: List of required Python libraries.
- **Background.png**: Background image used in the application.
